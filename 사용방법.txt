1. main.py 실행
2. 메뉴 입력
3. 모든 메뉴 입력 완료 시 입력 버튼 클릭
4. 각 메뉴별로 재료 확인하며 수정이 필요하면 수정
5. 해당 메뉴의 재료 수정 완료 시 --> 버튼을 클릭하여 전체 재료 칸으로 넘김
6. 4~5번의 방법을 반복하며 모든 메뉴의 재료를 전체 재료칸에 저장함
7. 급식 사진과 간식 사진을 컴퓨터에서 선택(선택된 사진의 경로가 화면에 출력되면 정상적으로 돌아가는 것)
8. 간식칸과 파일 이름칸에 내용 입력
9. 생성 버튼을 클릭하여 한글파일 생성
**모든 입력은 아래 형식에 맞게 입력**

메뉴 입력은 줄바꿈을 기준으로 한개씩 입력   ex)쌀밥
                                            어묵국

간식 입력시 공백을 기준으로 나누기          ex)떡볶이 요플레

파일 이름은 YYDDMM요일 형식으로 작성        ex)220506금

*dll파일 설정하면 보안 팝업창 안뜸*
